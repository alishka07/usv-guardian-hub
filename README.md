# USV Guardian Hub

Платформа управления автономными надводными аппаратами (USV — Unmanned Surface Vehicle) для мониторинга качества воды на Казахстан.
---
## Стек

- **Vite** + **TanStack Start** (SSR-ready, file-based роутинг через `@tanstack/react-router`)
- **React 19** + **TypeScript 5.8**
- **Tailwind CSS 4** + **shadcn/ui** 
- **Recharts** — графики
- **Sonner** — toast-уведомления
- **lucide-react** — иконки
- **Cloudflare Workers** (`@cloudflare/vite-plugin`, `wrangler`) — деплой
- **Bun** — менеджер пакетов и dev-runtime

---
## Запуск

```bash
# установка зависимостей
npm install
# dev-сервер (Vite + HMR)
npm run dev
```

После запуска `npm run dev` приложение доступно на `http://localhost:3000`
